Theorie Arduino Hardware Aufbau eines Arduino UNO [8] Software Der Arduino wird mit C++ programmiert IDE Arduino IDE Die Arduino IDE ist eine Entwicklungsumgebung für Arduino basierte Mikrocontroller. PlatformIO PlatformIO ist eine Erweiterung für Visual Studio Code, welche die Entwicklung für Arduino basierte Mikrocontroller vereinfacht. Dies ist insbesondere für größere Projekte von Vorteil, da die IDE eine bessere Strukturierung der Projekte ermöglicht. Ebenso profitiert man von den Features und Erweiterungen von Visual Studio Code.
